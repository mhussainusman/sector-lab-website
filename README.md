# Sector//LAB 

A futurism, Y2K and streetwear e-commerce website concept designed for the Pakistani market.

> *"Crafting the streetwear that Pakistan deserves."*

## Live Design
[View Figma Prototype](https://www.figma.com/site/aBMepYJwjClT4pxRWU7ztP/Untitled?node-id=0-1&t=dCJf2XpzkcXNq0wO-1)

---

## Pages

| Page | Description |
|------|-------------|
| Main Page | Hero section + Best Sellers grid |
| Store Page | Product grid with sorting — Winter XV New Arrivals |
| Product Page | Product details, size selector, color picker, reviews |
| Cart | Items in cart with selected items summary and total |
| Checkout | Delivery info form + multiple payment options |
| Login | Email/password login + Google Sign-in |
| About Us | Brand story, mission, values, and team |
| Contact & FAQ | Contact form + expandable FAQ section |

---

## ✨ Features

- 🧭 Sticky header with hamburger navigation menu
- 🛒 Shopping cart with item selection and order summary
- 💳 Checkout flow with delivery form and payment options (Cash on Delivery, Credit/Debit Card, PayPal, Apple Pay)
- 🔐 Login page with Google Sign-in and email/password auth
- ⭐ Product ratings and customer reviews
- 📦 Product detail pages with size guide, color variants, and stock status
- ❓ FAQ section with expandable accordion answers
- 📬 Contact form with brand contact info
- 🌑 Consistent dark-themed footer across all pages

---

## Design System

**Typography**
- `IBM Plex Mono` — Titles and headings
- `Manrope` — Paragraphs and body text
- `Inter` — Certain UI elements
- `Michroma` — Logo, reviews, sidebar

**Color Palette**
| Color | Hex |
|-------|-----|
| White | `#FFFFFF` |
| Light Gray | `#F0F0F0` |
| Black | `#0C0C0C` |
| Blue (Accent) | `#2C78D4` |

**Layout**
- 12-column grid system for visual balance
- Sticky header, consistent footer
- Each page has a unique layout structure
